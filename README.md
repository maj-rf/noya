# Noya: A Stella Sora Team Builder

A Web App to export team builds as image and easily share it with other players.

## Tech & Tools

- React + TypeScript
- TailwindCSS
- Shadcn UI
- Tanstack Router
- Cloudinary

## Contributing

Feel free to report issues, suggest improvements, and create pull requests. I will try my best to fix bugs and implement new & neccessary features.
